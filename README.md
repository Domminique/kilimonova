## KilimoNova Agentic AI for Small holder Resilience

Smallholder farmers in Western Kenya face Triple Threat: Climate shifts, devastating pests and fragmented markets . KilimoNova turns a basic Kes 2000 feature phone into a PHD-level agronomist and procurement officer using Agentic AI

## Key features
* Plant Doctor - instant Swahili pest Diagnosis USSD
* Autonomous Action - The agent doesnt just talk, it calls tools to alert extension officers via SMS
* Integrated payment - One-click payments

## Tech Stack

* Brain : Amazon Nan0-Micro via amazon bedrock
* Runtime- AWS Lambda (Python 3.12)
* Gateway : africa's Talking USSD and SMS API
* Monitoring : AWS CloudWatch logs Insight



  ## Quick Start
  * Clone https://github.com/Domminique/kilimonova/edit/main/README.md
  * AWS setup : Enable Amazon Nova-Micro and deploy Lambda
  * Env variables, your AT_API_KEY
  * USSD , connect your AFrica's Talking callback URL to the Lambda API Gateway
